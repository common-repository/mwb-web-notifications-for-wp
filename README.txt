== Frequently Asked Questions ==

= Can the admin show notifications to their users every 3 hours and also change the timing according to their need? =
 
 Yes, they can. To schedule the notifications, go to General settings>> Enter Time Span Difference to Show A Notification. And set it according to their needs.

= Can we show the notification to our guest users too? =
 
  Yes, you can show the notification to guest users also. For this, please go to the plugin settings>> General settings and enable show notification to guest users.

= How many types of predefined notifications have you provided in your plugin? =

 We have provided nine types of predefined notifications in our plugin.

= Can we create our custom type of notification? =

  Yes, the admin can create a custom notification. For this, go to the Notification tab >> Add new >> Choose notification type >> Custom.

= Is there any option so that the admin can choose the notification position and notification animation direction according to their need? =

   Yes, the admin can choose the notification position and notification animation direction from the notification edit page.

= Can we set the notification entry according to our needs? =

Yes, you can set the entry effect also. For this, Go to the Notification edit page, choose the entry effect, and save your changes. 
 
= Is there any option so that the admin can choose the notification trigger event? =

   The admin can choose the event at which notification will appear. There are four predefined trigger events to choose from: Page Load, Exit Intent, After Few Seconds of Page Load, and After Some Scroll On Page. 

For this, please Go to the Notification edit page >> Choose an event to show a notification.

= Can the admin create unlimited notification by using MWB Web Notifications for WP Plugin? =

   Yes, our notifications plugin allows the admins to create as many notifications as they need for their WordPress or WooCommerce website. You can create multiple notifications from the given nine notification types
